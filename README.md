# readable-ui-system-by-typescript
メニュー画面表示をTypeScriptで実装しました。

# 実現方法
chatGPTにSHOP.ERBをTypeScriptに変換してもらい、指示どおりコンパイルした。

# 苦労した点
## ローカルサーバ経由で開かないとScriptの実行をブロックされてしまう点
コードを睨んだが分からず、再びchatGPTに聞いて解決した。

# 今後の課題
## TypeScriptの構文自体を把握していく
まだAI頼りでコードを書いただけなので、可読性やメンテナンス性に不安があり、微修正も効率が悪いと考えられる。  
今後は(機会があれば)言語自体の学習も行っていきたい。

# 動作確認方法
1.コマンドプロンプトで「npx serve」を実行し、ローカルサーバーを立てる。  
2.ローカルサーバを立てた時に表示されたアドレス経由で「index.html」を開く。  
3.「表示例.png」のような画像が表示される。
