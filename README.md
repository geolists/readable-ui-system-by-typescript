# readable-ui-system-by-typescript
メニュー画面表示をTypeScriptで実装した。

# 実現方法
SHOP.ERBをTypeScriptで実装した。

# 苦労した点
## ローカルサーバ経由で開かないとScriptの実行をブロックされてしまう点
コードを睨んだが分からず、chatGPTに聞いて解決した。

# 今後の課題
## TypeScriptの構文自体を把握していく
まだ可読性やメンテナンス性に不安があり、微修正も効率が悪いと考えられる。  
今後は(機会があれば)言語自体の学習も行っていきたい。

# 動作確認方法
1.「index.html」「main.js」「style.css」をダウンロードし、同じフォルダに入れる。
2.コマンドプロンプトでカレントディレクトリを1.で作ったフォルダ、またはその直上にする。
3.「npx serve」を実行しローカルサーバーを立てる。  
4.ローカルサーバを立てた時に表示されたアドレス経由で「index.html」を開く。  
5.「表示例.png」のような画像が表示される。
